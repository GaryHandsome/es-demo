函数.prototype - 对象{} - 原型对象

使用原型对象，可以扩展属性和方法

有以下两种扩展方法
1）逐个扩展
    构造方法名称.prototype.属性名称 = 值 ;


    构造方法名称.prototype.方法 = function(){
        // ....
    } ;

2）统一扩展（完全重写原型） - 把属性和方法统一定义
    构造方法名称.prototype = {
        constructor:构造方法名称,
        属性名称：值 ,
        ...

        方法名称：function() {
            ...
        }
        ...
    }

    注意：使用统一扩展时，构造方法实例的对象中，该对象的 constructor 属性就丢失了，一般我们显式把 constructor 重新指定为构造方法

原型链
    把各个原型串联起来形成链条，称之为原型链

    任何对象都有一个__proto__属性，此属性存储的是构造此对象的构造方法的prototype

    查找 - 访问对象的属性和方法
